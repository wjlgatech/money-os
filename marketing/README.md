- **Budget**: $0 (bootstrapped, all organic)

## Execution Order

### Phase 1: Foundation (Weeks 1-2)
1. Deploy landing page
2. Set up email capture (ConvertKit/Buttondown — free tier)
3. Create social profiles (Twitter, LinkedIn)
4. Optimize plugin marketplace listing

### Phase 2: Seed (Weeks 3-4)
1. Post Reddit content (r/ClaudeAI first, then r/personalfinance, r/financialindependence)
2. Launch Twitter thread
3. Start "Build in Public" series
4. Begin email sequence for early signups

### Phase 3: Amplify (Weeks 5-8)
1. Product Hunt launch (Tuesday, target top 5)
2. Start blog series (2 posts/week)
3. Launch Money Myths video series
4. Outreach to creators and influencers

### Phase 4: Compound (Weeks 9-12)
1. Share card virality should be generating organic installs
2. Community building (Discord or dedicated subreddit)
3. User testimonials and case studies
4. Iterate based on dashboard data

## The Viral Engine
```
Freedom Calculator (free, no signup, 60 seconds)
    ↓ "This is cool, what else?"
Landing Page → Email Capture
    ↓ 7-email nurture
Plugin Install
    ↓ Use skills → Generate share cards
Screenshot & Share on Social
    ↓ Friends see → "What's YOUR Freedom Number?"
Freedom Calculator (loop restarts)
```
