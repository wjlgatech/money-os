Run the tax-strategy skill to generate a comprehensive, year-round tax optimization plan.

Gather from the user (if not already known):
1. Filing status (single, MFJ, MFS, HoH)
2. Approximate gross income and income sources (W-2, self-employment, investments)
3. Tax bracket (or estimate from income)
4. Account types held (401k, Roth IRA, Traditional IRA, HSA, taxable brokerage)
5. Any equity compensation (RSUs, ISOs, NSOs, ESPP)
6. State of residence

Then run all 7 capability checks from the tax-strategy skill and produce:
- Bracket management analysis
- Asset location recommendations
- Roth conversion opportunity analysis
- Investment decision tax modeling
- Equity compensation strategy (if applicable)
- Advanced strategy screening (QSBS, QOZ, DAF, Mega Backdoor Roth, NUA, HSA max)
- Year-end tax checklist

Produce both a summary in conversation and an Excel spreadsheet for multi-year planning.
